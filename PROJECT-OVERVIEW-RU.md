# Vocabify — Обзор проекта

## 1. Идея

Vocabify — браузерное расширение для Chrome, которое превращает обычный серфинг в процесс изучения иностранных слов. Пользователь выделяет незнакомое слово на любой веб-странице, получает мгновенный перевод и сохраняет в личный словарь. Система интервального повторения (SRS) автоматически напоминает слова в нужный момент, чтобы они закрепились в долговременной памяти.

**Ключевая проблема:** Люди читают тонны контента на английском, встречают незнакомые слова, но не запоминают их — нет удобного инструмента, который бы бесшовно интегрировался в процесс чтения.

**Решение:** Расширение, которое работает прямо в браузере, не требует переключения контекста и превращает пассивное чтение в активное обучение.

---

## 2. Видение (Vision)

Стать главным инструментом для изучения лексики через реальный контент. Не заменять Duolingo или учебники, а дополнять их — работать с тем, что пользователь уже читает: статьи, документация, Reddit, YouTube, новости.

**Долгосрочная цель:** платформа для контекстного обучения языкам, которая:
- Знает уровень пользователя (A1–C2) и адаптируется
- Подсказывает слова в контексте реальных текстов
- Использует AI для объяснений и упрощения текстов
- Работает на всех сайтах, включая YouTube (субтитры)
- Геймифицирует процесс (XP, уровни, ачивки, стрики)

## 3. Что уже реализовано

### 3.1 Захват и перевод слов
- Выделение слова/фразы на любой странице — всплывающее окно с переводом
- Контекстное меню (правый клик — "Translate" / "Save to Vocabify")
- Горячая клавиша Ctrl+Shift+T
- Автоматическое извлечение примера из окружающего текста
- Обогащение слова: синонимы, антонимы, определения, фонетика
- Детекция фразовых глаголов (100+) и коллокаций (55+)
- Поддержка выделения фразы целиком
- Лемматизация (running → run, carried → carry) — предотвращает дубликаты

### 3.2 Интервальное повторение (SRS)
- Тост-уведомления каждые 30 минут с карточкой слова (настраивается)
- Интервалы: 1, 3, 7, 14, 30, 90 дней
- Статусы: new → learning → known
- Система сложности слова
- Учёт забытых слов
- Режим "Не беспокоить" и лимит напоминаний в день

### 3.3 Дашборд (новая вкладка)
- **Обзор** — статистика, слово дня, быстрый доступ
- **Повторение** — полноценная сессия review с приоритизацией слов
- **Словарь** — список всех слов с сортировкой, фильтрацией, поиском, пакетным удалением
- **Сложные слова** — отдельный список проблемных слов
- **Статистика** — подробная аналитика

### 3.4 Режимы обучения
- **Quiz Mode** — тесты с множественным выбором (слово→перевод, перевод→слово)
- **Writing Practice** — ввод перевода с клавиатуры или голосом
- **Слово дня** — ежедневная карточка с приоритетом на сложные/забытые слова

### 3.5 Аналитика и инсайты
- Тепловая карта активности (90 дней, стиль GitHub)
- Тренд точности (30 дней — запомнил/забыл)
- Распределение силы слов (слабые / растущие / сильные / освоенные)
- Оценка уровня CEFR (A1–C2)
- Стрики (текущий и рекордный)
- Топ-5 сложных и самых повторяемых слов
- Скорость обучения, retention rate

### 3.6 Геймификация
- Система XP за действия (сохранение, повторение, стрики)
- 15+ уровней прокачки
- 15+ ачивок с уведомлениями
- Ежедневная цель XP

### 3.7 Анализ страниц
- Оценка сложности страницы (CEFR A1–C2)
- Процент понимания текста
- Список незнакомых слов с группировкой по уровню
- Боковая панель с анализом

### 3.8 AI-функции
- AI-объяснение слова в контексте
- AI-упрощение текста до нужного уровня
- Лимиты: 1 вызов/день бесплатно, 10/день для Pro

### 3.9 YouTube
- Извлечение субтитров/транскрипций из видео
- Выделение и перевод слов из субтитров

### 3.10 Импорт/Экспорт
- Экспорт в JSON (полный бэкап), CSV, формат Anki
- Импорт из всех трёх форматов
- Пропуск дубликатов

### 3.11 Прочее
- Vocabulary Radar — отслеживание повторяющихся незнакомых слов (≥4 раза за 7 дней → предложение сохранить)
- Бейдж на иконке расширения с количеством слов
- Кнопка Undo при сохранении (5 секунд)
- Прогрессивные подсказки для новых пользователей
- Озвучка слов (Text-to-Speech)

## 4. Целевая аудитория

### Основная
- **Люди, изучающие английский** (уровень A2–B2), которые много читают в интернете
- Возраст: 18–40
- География: СНГ, Латинская Америка, Азия, Восточная Европа
- Мотивация: работа, эмиграция, учёба, IELTS/TOEFL

### Вторичная
- Полиглоты, изучающие несколько языков
- Студенты языковых вузов
- Преподаватели (для рекомендации ученикам)
- Разработчики, читающие англоязычную документацию

## 5. Конкуренты

| Продукт | Плюсы | Минусы | Наше преимущество |
|---------|-------|--------|-------------------|
| **Google Translate (расширение)** | Бесплатно, быстро | Не запоминает слова, нет SRS | Мы сохраняем и повторяем |
| **Toucan** | Inline-переводы | Нет SRS, нет квизов | Полная система обучения |
| **Rememberry** | SRS + расширение | Устаревший, мало фич | Больше режимов, AI, аналитика |
| **Language Reactor** | YouTube/Netflix фокус | Только видео | Работаем на любом сайте |
| **Anki** | Мощная SRS | Нужно вручную создавать карточки | Автоматический захват из контекста |
| **Duolingo** | Геймификация | Искусственные тексты | Реальный контент пользователя |

**Уникальное преимущество Vocabify:** бесшовная интеграция в процесс чтения + автоматический захват контекста + полная SRS + AI-объяснения + геймификация — всё в одном расширении.

## 6. Монетизация — как зарабатывать

### 6.1 Текущее состояние
Реализована Pro-gate система (ограничение AI-вызовов: 1/день бесплатно vs 10/день Pro), но **платёжная система ещё не подключена**.

### 6.2 Модель Freemium (основной план)

**Бесплатно (Free):**
- Безлимитное сохранение слов
- Полная SRS-система
- Quiz и Writing Practice
- Базовая статистика
- Импорт/экспорт
- 1 AI-вызов в день

**Pro ($3–5/мес или $29–39/год):**
- 10+ AI-объяснений в день
- AI-упрощение текстов
- Продвинутая аналитика (тренды, прогнозы)
- Приоритетная поддержка
- Синхронизация между устройствами (будущее)
- Персонализированные рекомендации слов

### 6.3 Источники дохода

#### A. Подписка (Subscription) — основной
- Месячная ($3–5) или годовая ($29–39)
- Интеграция через Stripe или Paddle
- Оплата через сайт, активация по email/коду в расширении

#### B. Разовая покупка (Lifetime)
- $19–39 за пожизненный доступ
- Привлекает тех, кто не любит подписки
- Хорошо работает на ProductHunt / AppSumo

#### C. Донаты / "Buy me a coffee"
- Для начального этапа, пока нет Pro
- Кнопка в настройках расширения

#### D. Образовательные лицензии (B2B)
- Пакеты для школ/курсов/компаний
- Общий дашборд для преподавателя
- Отслеживание прогресса группы

#### E. Affiliate / партнёрство
- Рекомендации языковых курсов (iTalki, Preply)
- Партнёрство с сервисами вроде Anki, Quizlet

#### F. Premium-контент
- Готовые наборы слов по темам (бизнес, IT, медицина)
- Тематические квизы
- Курируемые списки слов от преподавателей

## 9. Риски и как их минимизировать

| Риск | Вероятность | Решение |
|------|-------------|---------|
| Блокировка undocumented Translate API | Высокая | Fallback-цепочка из 3 сервисов, свой прокси |
| Низкая конверсия в Pro | Средняя | A/B тесты ценообразования, больше Pro-фич |
| Chrome Web Store отклоняет | Средняя | Privacy Policy готов, минимальные permissions |
| Конкурент скопирует | Низкая | Быстрая итерация, community, уникальные AI-фичи |
| Рост цен на AI API | Низкая | Кэширование, переход на open-source модели |
| Пользователи не возвращаются | Средняя | Стрики, email-ремайндеры, Telegram-бот |

## 10. Сводка

**Vocabify** — полноценная платформа для изучения лексики через реальный контент в браузере. Уже реализовано 20+ фич: от перевода до AI-объяснений, квизов, SRS и геймификации. Проект готов к публикации в Chrome Web Store и запуску монетизации через Freemium-модель.

Основной источник дохода — Pro-подписка ($3–5/мес) с расширенными AI-возможностями и продвинутой аналитикой. Долгосрочная стратегия — стать незаменимым инструментом для всех, кто учит язык через чтение, с расширением на мобильные платформы, мультиязычность и B2B-сегмент.
